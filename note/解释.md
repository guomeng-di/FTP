# 6
```c++
void send_response(int fd,const string& s){
    int sent=0;
    const char* buf=s.c_str();
    int len=s.size();
    while(sent<len){
        int n=send(fd,buf+sent,len-sent,MSG_NOSIGNAL);
        if(n<=0) break;
        sent+=n;
    }
    return ;
}

MSG_NOSIGNAL:
当不写时,客户端断开,则服务器也无法send,会直接退出崩溃

作用：防止客户端异常断开 → 导致服务器崩溃！
```
